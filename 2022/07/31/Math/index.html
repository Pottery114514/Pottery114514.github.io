极限&无穷小&无穷大(infinity) (翻译)

为什么我们需要极限(Limits)和无穷小(Infinitesimals)？许多数学课程在没有任何前兆的情况下引入极限,无穷小和极小数字（TM）。但我们为什么在意呢？

极限还是无穷小？

从逻辑上讲，这两种方法都解决了“零和非零”的问题。我喜欢无穷小，因为它们允许“另一个维度”，这似乎比“总是在你力所能及的范围内”更精准。无穷小是微积分直觉的基础，并出现在物理学和其他使用它的学科中。

1.零的悖论(The Paradox of Zero)
将曲线分解成矩形有一个问题：我们如何获得如此薄的切片，以至于注意不到它们，但又大到足以“存在”？如果切片太小而无法被注意到（零宽度），则模型看起来与原始形状相同（我们看不到任何矩形！)这没有任何好处,“简单”模型与原始模型一样复杂！此外，将零宽度切片相加不会让我们取得任何进展。如果切片很小但可以测量，则幻觉就会消失。我们会看到我们的模型是锯齿状的，但并不准确。数学家该怎么办？

我们想要两者中的佼佼者：薄到无法看到它们的切片（对于准确的模型），以及足够厚的切片以便创建更易于分析的模型。一个两难的境地就在眼前！(A dilemma is at hand!)

2.极限和无穷小概述
让我们看看两种方法如何将曲线分解为矩形.极限：“给我你的精度（我知道你有一个，你是有限的，不完美的人类！），我会给你画一条曲线。你尺子上的最小精度是多少？英寸？好吧，我会给你画一个毫米级的阶梯曲线，而你永远不会知道。你有一把毫米尺，是吗？我将以纳米为单位绘制曲线。无论你的准确性如何，我都更精确,你永远不会看到锯齿。无穷小：有一个完整的无限小的维度，我将在那里制作曲线。精确度完全超出了你的范围——我在亚原子水平，你是一个穴居人，几乎不能走路和嚼口香糖。这就像从真实的平面到达想象中的平面一样——你就是做不到。对你来说，我在亚原子水平上制作的矩形是你见过的最完美的曲线。

极限停留在我们的维度上，但具有“恰到好处”的准确性来维持完美模型的错觉。无穷小在另一个维度上构建模型，它在我们的维度上看起来非常准确。这两种方法的诀窍在于，模型是构建在我们的准确性水平之上的。我们可能知道模型是锯齿状的，但我们无法区分(can’t tell the difference) - 我们所做的任何检测都显示模型和真实项目是相同的。

3.在另一个维度上工作—无穷小
在使用简化模型(simplified model)进行推理时，我们需要小心。我们需要在更高的准确度水平上“做我们的工作”，并将最终结果带回我们的项目。否则，我们将丢失信息。

假设一个虚数 （i） 游玩实数行。每个人(实数)都认为他是零.毕竟，Re（i） = 0。但是他(i)做了一个把戏:“Square me！”他说，他们(实数)确实照他说的做了：“i * i = -1”，所有数字都感到惊讶。

对于实数们，这意味着“0 * 0 = -1”，一个巨大的悖论(或许是对实数数学家们的当头一棒)。

但他们的困惑是从他们的角度产生的,他们只认为这是“0 0 = -1”。是的，Re（i） Re（i） = 0，但这不是他(i)的操作！Re（i * i），这是完全不同的！i在自己的维度平方，并将这个结果带回到实数的维度。我们需要平方i，虚数，而不是0.当心微积分中的类似错误：我们处理对我们来说看起来像零的微小数字，但我们不能假设它们是零（就像实数们对待i像0一样）。不，我们需要在另一个维度上“做数学运算”，并将结果带回来。

极限和无穷小对如何完成此转换有不同的方式(perspectives)：
极限：“进行数学运算”，精度水平刚好超出您的检测范围（毫米），并将其恢复为体重秤上的数字（英寸）.无穷小：“运算”在不同的维度，并将其带回“标准”维度（就像取复数的实部一样;你取超实数的“标准”部分 - 之后更多）.从来没有人告诉我微积分可以让你以更高的精度水平工作，使用更简单的模型，并将结果带回我们的世界。

4.一个真实的例子：$\frac{sin(x)}{x}$
让我们尝试一个概念性示例。假设我们想知道$\frac{sin(x)}{x}$在x=0时会发生什么。现在，如果我们只考虑x = 0，我们会得到一个荒谬的结果：0 / 0，这可能是任何东西。让我们退一步：“x = 0”在我们的世界中意味着什么？好吧，如果我们允许存在更高水平的准确性，我们知道这一点：看起来为零的事物在不同的维度上可能是非零的（就像i对我们来说可能是0，但不是）.我们要说的是，在这个更高的精度水平上，x可以非常非常接近于零，但不是“真正的零”。直观地说(Intuitively)，您可以将x视为0.0000…00001，其中“…”为足够多的零.在极限术语中，我们说x = 0 + d（delta，一个很小的变化，使我们保持在误差范围内），在无穷小的术语中，我们说x = 0 + h，其中h是一个微小的超实数，称为无穷小.

好吧，我们有x在“对我们来说为零，但不是真的”的感觉。现在我们需要一个更简单的 sin（x） 模型。为什么？好吧，正弦是一个疯狂的重复曲线，我们很难知道发生了什么。但事实证明，直线是描述短距离曲线的好模型：

就像我们可以将填充形状分解成小矩形以使其更简单一样，我们可以将曲线解剖为一系列线段。在0左右，sin（x）看起来像x。因此，我们用“x”替换sin（x）。新的比率是多少？

好吧，“x/x”是1。请记住，我们并没有真正被零除，因为在这个超精确的世界中,x很小但非零（0 + d，或0 + h）。当我们“取极限或”取标准部分“时，这意味着我们进行数学运算（x / x = 1），然后找到我们世界中最接近其的数字（1）。所以，1是当sin（x）/ x接近零时我们得到的 - 也就是说，我们使x尽可能小，所以它对我们来说变成0(事实上并不是)。如果x变成零，那么这个比率将是未定义的.因此，据我们所知，“sin（x）/x”看起来像“x/x = 1”。

5.Warning：诀窍并不总是有效
有些函数真的很“跳跃”——它们可能在无穷小的层面上有所不同。这意味着我们无法可靠地将结果带回我们的世界。如$sin\frac{1}{x}$

看起来该函数在微观层面上不稳定，并且行为不“平稳”。限制的严格部分是弄清楚哪些函数表现得足够好，可以制作简单而准确的模型。幸运的是，世界上大多数自然函数（x，x2， 罪， ex） 表现良好，可以用微积分建模。

独立事件的概率—-“至少”

教科书作者（和测验作者）喜欢这种类型的问题，所以值得一提。“至少”问题通常希望您将几个概率规则组合在一起.

示例 ：你正在计划一个烧烤，你想在下午2点开始。弗雷德和乔住在城镇的两边，他们都同意带上木炭。问题是他们都是懒惰者,弗雷德有40%的可能迟到，乔有30%的可能迟到。您在下午2点之前开始烧烤的概率是多少？

解决方案：这是独立事件的一个“至少”问题，尽管这次独立事件的概率不同。要在下午2点之前吃木炭烧烤，就至少有一个人必须在那时出现。至少有一个人准时到达的概率是多少？同样，你可以计算出他们俩准时的概率，弗雷德准时但乔迟到的概率，以及弗雷德的迟到和乔准时的概率 — 所有这些加在一起就是准时的概率。但同样,补充是你的朋友. “准时木炭”的补集是“木炭迟到”，只有当它们都迟到时才会发生这种情况。

P(准时) = 1-P(迟到) = 1-P(Mr.fred&Jo都迟到)

请记住,补集(complement)是你的朋友.

小心你的逻辑！你真的需要一步一步地完成工作，并写下你的步骤。请记住，所有都不是(all are not)不同于不是所有都是(not all are)。

条件概率 P(B|A)

2012年，本田雅阁是美国最常被盗的车辆. 这是否意味着您的本田雅阁比其他车型更容易被盗？

你接受了一种罕见的流感病毒的检测，结果是阳性的。你的医生告诉你测试是99%准确的。这是否意味着你有99%的机会感染了这种流感病毒？

在纽约市，强奸受害者识别出仅与0.0001%的人匹配的身体特征。警方找到具有这些特征的人并逮捕了他。难道他只有0.0001%的几率是无辜的吗？

这些是条件概率的例子,指一个事件发生的情况下另一件事发生的概率.

离散概率模型

  • 离散型随机变量

变量是“在进行实验或进行观察时测量或观察到的特征”。

您随机选择 12 个下北泽学生，测量他们的身高，然后取平均值。“学生身高”是一个连续随机变量，“12 名学生样本中的平均身高”是另一个连续随机变量。

您随机选择40个家庭，并询问每个家庭中的孩子数量。“家庭中的孩子数”是一个离散的随机变量，而“40个家庭样本中的平均孩子数”是一个连续的随机变量。
离散可以理解为非均匀分布。

  • 分布列

  • 离散概率分布
    1.均值和标准差
    定义：假设你做了很多次概率实验。离散概率分布的均值（μ）是来自无限大量试验的结果的平均值，离散概率分布的标准差（σ）是来自无限大量试验的结果的标准差。任何概率分布的平均值也称为预期值，因为从长远来看，它是预期的平均结果。
    总之,一般来说，均值告诉你长期的平均结果，SD告诉你任何特定试验的不可预测性。
